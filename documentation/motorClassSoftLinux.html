EPICS Training 2015

Using Motors & Motor Control Internals


The following Linux software that is required to run the IOC that will be used in the class.

X-server

Most Linux distributions come with X servers.

MEDM

There isn't an easy way to provide a MEDM binary for linux, due to library issues.
APS-maintained machines should be able to run MEDM from /APSshare after sourcing one of the following scripts

bash:
$ source /APSshare/epics/startup/epicsSetup.profile

tcsh:
$ source /APSshare/epics/startup/epicsSetup.cshrc

Python

Python 2.7 is required to run the virtual motor controller server. Many Linux distributions ship with older versions of python.

APS-maintained machines can run anaconda python from /APSshare:
/APSshare/anaconda/x86_64/bin/python

Linux machines without access to APSshare can install anaconda locally (download reqruires giving continuum.io your email address):
https://store.continuum.io/cshop/anaconda/
