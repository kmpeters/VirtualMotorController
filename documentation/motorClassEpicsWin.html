EPICS Training 2015

Using Motors & Motor Control Internals

Download the IOC

The IOC can be downloaded from here: vmc.zip

Install the IOC

Unzip the IOC anywhere on your system where it is convenient. This will create a "vmc" directory.

Configure the IOC

You should only need to modify one line of one file:
  • Replace "vmc" with your initials on line 4 of ENV_WINDOWS.bat
    • This changes the PV prefix, which is necessary to avoid PV name collisions.

    Run the Virtual Motor Controller

    Double-click: start_vmc1_windows.bat

    Start MEDM

    Double-click: start_medm_windows.bat

    Start the IOC

    If you're running 32-bit Windows, double-click: start_ioc_win32-x86.bat
    If you're running 64-bit Windows, double-click: start_ioc_windows-x64.bat
