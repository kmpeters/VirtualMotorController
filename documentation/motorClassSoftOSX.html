EPICS Training 2015

Using Motors & Motor Control Internals


The following OS X software that is required to run the IOC that will be used in the class.

X-server

An X-sever is required for MEDM. APS-maintained machines may already have XQuartz installed.

If XQuartz isn't already installed, download and install it:
http://xquartz.macosforge.org/landing/

MEDM

A version of MEDM for OS X can be downloaded here: medm-mac.zip

Unfortunately this version expects to find openmotif libraries in /opt/local/lib, which is where MacPorts installs them:
$ tool -L medm
