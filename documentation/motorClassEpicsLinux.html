Using Motors & Motor Control Internals

Download the IOC

The IOC can be downloaded from here: vmc.zip

Install the IOC

Unzip the IOC anywhere on your system where it is convenient. This will create a "vmc" directory.

Configure the IOC

You should only need to modify one line of one file:
  • Replace "vmc" with your initials on line 4 of ENV_POSIX
    • This changes the PV prefix, which is necessary to avoid PV name collisions.

    Run the Virtual Motor Controller

    $ cd vmc
    $ ./start_vmc1_posix

    Start MEDM

    $ ./start_medm_posix

    Start the IOC

    $ ./start_ioc_linux-x86_64

    Note: the IOC was compiled on RHEL6
