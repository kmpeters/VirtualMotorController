EPICS Training 2015

Using Motors & Motor Control Internals


The following Windows software that is required to run the IOC that will be used in the class.

X-server

An X-sever is required for MEDM. APS-maintained machines are likely to already have Exceed installed.

If Exceed isn't already installed, download and install Xming:
http://www.straightrunning.com/XmingNotes

EPICS tools for Windows

EPICS tools for Windows provides MEDM and caRepeater, among other things:
http://www.aps.anl.gov/epics/distributions/win32/index.php

Python

Python 2.7 is required to run the virtual motor controller server:
https://www.python.org/downloads/release/python-279/

Or just use this pre-packaged executable (unzip it into the IOC dir): server-win.zip
